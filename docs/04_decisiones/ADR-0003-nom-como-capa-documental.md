# ADR-0003: NOM como capa documental

## Estado

Aceptado.

## Contexto

Cumplir NOM es obligatorio, pero la consulta no debe sentirse como una plantilla normativa. La atención clínica requiere conversación, exploración de contexto, motivación, barreras, interpretación y acuerdo terapéutico.

## Decisión

La NOM será una capa documental que organiza datos generados por el flujo MetaboCare.

## Consecuencias

- El sistema futuro deberá poder generar documentos normativos sin alterar el flujo humano de la consulta.
- La documentación NOM se alimentará de formatos operativos y datos clínicos capturados durante la atención.
- La prioridad de diseño seguirá siendo el flujo clínico-operativo.
