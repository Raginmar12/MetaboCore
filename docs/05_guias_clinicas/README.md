# Guías clínicas base

Esta carpeta reúne guías clínicas de referencia para MetaboCare. Su función es apoyar el criterio clínico, el lenguaje de consulta, el diseño de formatos y la interpretación operativa de los flujos documentados.

## Alcance

- Sirve como biblioteca clínica de apoyo para `docs/01_flujo_consulta/`, `docs/02_formatos/` y decisiones de diseño relacionadas.
- Ayuda a justificar criterios conversacionales, manejo de barreras, adherencia, estigma de peso y comunicación centrada en la persona.
- Puede alimentar manuales operativos internos ubicados en `docs/01_flujo_consulta/`, sin sustituir el flujo principal.
- No reemplaza el flujo humano principal de consulta.
- No es un formato clínico operativo.
- No es un schema ni contrato estructurado.
- No es capa normativa oficial ni declaración de cumplimiento NOM-004.
- No es expediente clínico, protocolo rígido, checklist obligatoria ni indicación terapéutica individual.

## Guías disponibles

- `entrevista_motivacional_metabocare.md`: referencia clínica base sobre entrevista motivacional aplicada a MetaboCare, obesidad, enfermedades cardiometabólicas, adherencia, barreras y lenguaje clínico no estigmatizante.

## Manuales operativos relacionados

- `docs/01_flujo_consulta/entrevista_motivacional_operativa.md`: placeholder pendiente para el futuro manual operativo interno de entrevista motivacional breve.

## Uso recomendado

Antes de modificar flujos, formatos o schemas relacionados con entrevista motivacional, adherencia, barreras, comunicación clínica, estigma de peso u objetivos de cambio, revise la guía clínica relevante de esta carpeta.

Las guías deben informar el diseño operativo sin convertir el repositorio en sistema de captura, expediente electrónico, protocolo rígido ni repositorio de datos reales de pacientes.
