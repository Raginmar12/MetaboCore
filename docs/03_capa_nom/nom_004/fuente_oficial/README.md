# Fuente oficial NOM-004

Esta carpeta contiene la transcripción en Markdown de la **NOM-004-SSA3-2012, Del expediente clínico**, utilizada como fuente normativa de referencia para MetaboCore.

Los archivos de esta carpeta deben tratarse como fuente primaria de consulta interna y no como formatos operativos.

La interpretación operativa, checklist, matriz de requisitos y guía de revisión arquitectónica deben vivir en los documentos Markdown superiores de `docs/03_capa_nom/nom_004/`.

> Nota: para interpretación jurídica o normativa exacta, consultar siempre la publicación oficial correspondiente.
