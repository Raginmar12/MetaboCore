# Núcleo conceptual

MetaboCare se enfoca en el manejo del eje cardiometabólico. El objetivo no es tratar diagnósticos aislados, sino intervenir sobre los procesos fisiopatológicos subyacentes que los conectan.

## Modelo central

Disregulación energética mediada por hiperinsulinemia crónica.

Este modelo permite interpretar múltiples manifestaciones clínicas como expresiones de un sistema metabólico desregulado, en lugar de abordarlas como problemas aislados sin relación entre sí.

## Procesos centrales

- Exceso de grasa corporal, especialmente visceral.
- Disfunción de insulina/glucosa.
- Dislipidemia aterogénica.
- Disfunción vascular.
- Disfunción hepática metabólica.

## Manifestaciones clínicas principales

- Sobrepeso/obesidad.
- Prediabetes/diabetes tipo 2.
- Hipertensión arterial.
- Dislipidemia.
- Hígado graso.
- Hiperuricemia/gota como manifestación asociada relevante.

## Estructura de interpretación

- **Núcleo:** procesos metabólicos centrales.
- **Moduladores:** sueño, estrés, sedentarismo, síndrome de ovario poliquístico u otros factores que perpetúan o aceleran la disfunción.
- **Condiciones asociadas:** alteraciones que acompañan el cuadro, pero no pertenecen necesariamente al eje principal.

## Principio clínico

No se tratan enfermedades aisladas. Se intervienen sistemas desregulados.
